# 07 - Files (Python)

```bash
pip install -r requirements.txt
python files.py
```

## What this shows

| API | What it does |
|---|---|
| `sandbox.write_file(path, content)` | Write a string or bytes |
| `sandbox.read_file(path)` | Read bytes back |
| `sandbox.stat_file(path)` | Size, is_directory, etc. |
| `sandbox.mkdir(path)` | Create a directory |
| `sandbox.list_files(path)` | List directory entries |
| `sandbox.delete_file(path, recursive=True)` | Remove file or directory |
